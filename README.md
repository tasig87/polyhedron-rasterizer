**How To Run**
1. Import main.py and polyhedron.py into a Python workspace.
2. If necessary, install the following relevant libraries into your workspace:
  - numpy
  - networkx
  - pygame
3. Run the "main.py" file in Python.
4. The files as is will show a spinning wireframe of the default "25W:TSC" Polyhedron mentioned in the documentation. Further edits will require editing the code in main.py.
